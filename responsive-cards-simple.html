How It Works

Layout Approach

Wrapper-Based Flexbox Design: Uses a dedicated scrollable wrapper container for mobile horizontal scrolling, ensuring only the cards section scrolls while the rest of the page remains fixed. Pure CSS flexbox implementation with no frameworks required.

Desktop Layout (> 768px)

  • .cards-wrapper: overflow: hidden (no scrolling needed)
  • .cards-container: display: flex; flex-wrap: wrap creates responsive grid
  • justify-content: center centers the grid on the page
  • Each card: width: calc(33.333% - 20px) for 3 cards per row
  • Minimum width: min-width: 280px prevents cards from getting too small

Mobile Layout (≤ 768px)

  • .cards-wrapper: overflow-x: auto - ONLY this container scrolls
  • .cards-container: width: max-content allows it to be wider than wrapper
  • flex-wrap: nowrap prevents wrapping to create horizontal layout
  • scroll-snap-type: x mandatory creates clean card alignment
  • Fixed card width: flex: 0 0 280px for consistent scrolling experience
  • Page remains fixed - only the cards section scrolls horizontally

Container-Based Scrolling Approach

  • Dedicated Wrapper: .cards-wrapper contains only the scrolling behavior
  • Isolated Scrolling: Only cards section scrolls - page remains fixed
  • Content Freedom: .cards-container can be wider than viewport
  • Precise Control: Scrolling is contained within the designated area
  • No Page Movement: Rest of webpage stays perfectly aligned

Mobile Layout Fixes

  • Viewport Containment: html, body { overflow-x: hidden } prevents page-level scroll
  • Wrapper Scrolling: .cards-wrapper { overflow-x: auto } isolates scroll to cards only
  • Container Width: width: max-content allows container to expand beyond wrapper
  • Proper Padding: Consistent 15px padding for perfect edge alignment
  • Button Contrast: Dark blue backgrounds with white text for accessibility
  • Touch Optimization: Proper touch targets and smooth gesture handling

Key Features

  • ✅ Smooth Scrolling: CSS-based smooth horizontal scrolling
  • ✅ Scroll Snapping: Cards snap to clean alignment points
  • ✅ Hidden Scrollbars: Clean mobile appearance
