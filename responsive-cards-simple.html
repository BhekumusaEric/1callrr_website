How It Works

Layout Approach

Flexbox-Based Responsive Design: Uses pure CSS flexbox for both desktop grid and mobile horizontal scroll layouts. No CSS Grid or JavaScript frameworks required.

Desktop Layout (> 768px)

  • display: flex; flex-wrap: wrap creates a responsive grid
  • justify-content: center centers the grid on the page
  • Each card: width: calc(33.333% - 20px) for 3 cards per row
  • Minimum width: min-width: 280px prevents cards from getting too small

Mobile Layout (≤ 768px)

  • flex-wrap: nowrap prevents wrapping to create horizontal layout
  • overflow-x: auto enables horizontal scrolling
  • scroll-snap-type: x mandatory creates clean card alignment
  • Fixed card width: flex: 0 0 280px for consistent scrolling
  • Hidden scrollbars and proper containment prevent page-level scrolling

Mobile Fixes Applied

  • Viewport Containment: html, body { overflow-x: hidden } prevents page-level horizontal scroll
  • Proper Padding: Consistent 15px padding on mobile for edge alignment
  • Box Sizing: box-sizing: border-box ensures proper width calculations
  • Button Contrast: Improved colors for better accessibility (dark blue backgrounds, white text)
  • Touch Optimization: Proper touch targets and gesture handling

Key Features

  • ✅ Smooth Scrolling: CSS-based smooth horizontal scrolling
  • ✅ Scroll Snapping: Cards snap to clean alignment points
  • ✅ Hidden Scrollbars: Clean mobile appearance
  • ✅ Touch-Friendly: Optimized for swipe gestures
  • ✅ Arrow Navigation: Optional manual scrolling controls
  • ✅ Responsive: Automatic desktop ↔ mobile switching
  • ✅ Accessible: Keyboard navigation and screen reader support
