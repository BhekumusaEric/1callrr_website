How It Works

Desktop Layout (Screens > 768px)

  • Cards display in a responsive grid using CSS Grid
  • grid-template-columns: repeat(auto-fit, minmax(280px, 1fr)) creates 3-4 cards per row
  • Cards have hover effects with smooth transitions
  • Navigation arrows are hidden on desktop

Mobile Layout (Screens ≤ 768px)

  • Cards switch to horizontal scrolling layout
  • display: flex; overflow-x: auto enables horizontal scrolling
  • scroll-snap-type: x mandatory creates clean snapping between cards
  • Scrollbars are hidden for a clean mobile look
  • Navigation arrows appear for manual scrolling control

Key Features

  • Smooth Scrolling: CSS scroll-behavior: smooth (implied by scroll-snap)
  • Scroll Snapping: Cards snap to position for clean alignment
  • Hidden Scrollbars: scrollbar-width: none and webkit scrollbar hiding
  • Touch-Friendly: Optimized for mobile swipe gestures
  • Responsive Design: Automatic switching between grid and scroll layouts

JavaScript Navigation (Optional)

The navigation arrows use JavaScript to smoothly scroll the container. Add this script for enhanced control:

document.addEventListener('DOMContentLoaded', function() {
    const container = document.getElementById('cardsContainer');
    const prevBtn = document.getElementById('prevBtn');
    const nextBtn = document.getElementById('nextBtn');

    // Scroll amount (card width + gap)
    const scrollAmount = 295; // 280px card + 15px gap

    prevBtn.addEventListener('click', () => {
        container.scrollBy({ left: -scrollAmount, behavior: 'smooth' });
    });

    nextBtn.addEventListener('click', () => {
        container.scrollBy({ left: scrollAmount, behavior: 'smooth' });
    });

    // Update button states based on scroll position
    function updateButtons() {
        const { scrollLeft, scrollWidth, clientWidth } = container;
        prevBtn.disabled = scrollLeft <= 0;
        nextBtn.disabled = scrollLeft >= scrollWidth - clientWidth - 1;
    }

    container.addEventListener('scroll', updateButtons);
    updateButtons(); // Initial state
});
